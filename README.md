# Material Measurement Fast Drawing
均均老闆詹法師電腦效能太弱，沒辦法跑繪圖軟體，所以我想幫他們用 Python 來繪圖
---
## 功能
1. 可以將檔案直接拖曳到執行檔上，當作 input 
2. 將量測結果做積分之後輸出圖表
3. 支援處理多份檔案

## 自行安裝方法
1. 撰寫完程式碼，使用 PyInstaller 包成執行檔即可執行
